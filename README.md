# Mentrix Landing Page

Professional, responsive landing page for Mentrix EdTech Platform built with React + Tailwind CSS.

## Features

✅ Hero section with email signup
✅ Exam showcase (WBCHSE, NEET, JEE, WBJEE)
✅ Feature highlights
✅ Transparent pricing
✅ FAQ section
✅ Mobile responsive
✅ Smooth scrolling navigation
✅ Professional design

## Setup & Development

### Prerequisites
- Node.js 16+ installed
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
